# 用户伙食预算规则

最后确认日期：2026-05-24

## 每日预算

| 餐别 | 预算 | 工作日（Mon-Fri） | 周末（Sat-Sun） |
|------|------|-------------------|----------------|
| 早餐 | ¥5 | 自费 ✓ | 自费 ✓ |
| 午餐 | ¥10 | **公司提供，免费** 🏢 | 自费 ✓ |
| 晚餐 | ¥10 | 自费 ✓ | 自费 ✓ |
| **每天自费合计** | | **¥15**（早+晚） | **¥25**（早+午+晚） |

## 代码中的计算逻辑

```python
def budget_per_day(d):
    return 15 if d.weekday() < 5 else 25  # 含早¥5
```

## 追踪范围

- **预算包含**：早餐¥5 + 午餐¥10 + 晚餐¥10（完整的三餐预算）
- **iCost 可追踪**：午餐 + 晚餐 + 零食饮料 + 水果（一级分类 == "餐饮"）
- **无法追踪**：早餐（iCost 无早餐分类）
- **对比方式**：餐饮实际支出 vs 含早的完整预算（早餐部分作为buffer）
